This project presents a **comparative study of first-order and second-order optimization methods** applied to convex and non-convex functions. The main objective is to understand how optimization algorithms minimize objective functions and analyze their convergence behavior. In this implementation, **Gradient Descent** represents the first-order method using only the gradient, while **Newton’s Method** represents the second-order method using both gradient and Hessian information. These algorithms are tested on mathematical functions to observe convergence speed, iteration behavior, and computational efficiency. The results show that Newton’s method generally converges faster than gradient descent for convex problems, while non-convex problems may contain multiple local minima. Visualization and performance comparisons are used to analyze algorithm behavior. The project also extends the study through future implementations including **multi-dimensional optimization**, **linear regression using gradient descent**, and **learning rate comparison experiments**. These extensions demonstrate how optimization techniques are applied in real machine learning scenarios. Overall, the project provides a practical understanding of optimization algorithms and their role in numerical optimization and machine learning.
